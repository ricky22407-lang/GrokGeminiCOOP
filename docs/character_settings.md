# 角色設定表 (Character Profiles)

## 1. 聖女 - 塞拉斯蒂亞 (Celestia)
*   **稱號**：光輝的虛偽
*   **外貌**：金髮碧眼，穿著高叉修女服（改造後），身材豐滿，表情常帶悲憫。
*   **性格**：表面聖潔無私，實則因背負過重期待而內心空虛。
*   **關鍵詞**：墮落、背德、信仰崩塌。
*   **心理路徑**：試圖感化主角 -> 發現神並不在乎她 -> 在主角的支配中找到釋放 -> 成為宣揚魔王教義的狂信徒。

## 2. 精靈女王 - 艾琳諾 (Elysia)
*   **稱號**：傲慢的翠玉
*   **外貌**：銀髮長直，尖耳，穿著極其暴露的藤蔓風格禮服，冷豔高貴。
*   **性格**：極度傲慢，看不起人類與魔族，自尊心極強。
*   **關鍵詞**：羞恥、高傲粉碎、公開處刑（心理層面）。
*   **心理路徑**：輕視主角 -> 戰敗被俘 -> 為了族人忍受屈辱 -> 自尊心轉化為對強者的絕對崇拜。

## 3. 王國女將軍 - 瓦爾基里 (Valkyrie)
*   **稱號**：折翼的赤龍
*   **外貌**：紅髮馬尾，小麥色肌膚，有肌肉線條，穿著破損的重甲與膠衣內襯。
*   **性格**：剛正不阿，重視榮譽與勝負，典型的騎士。
*   **關鍵詞**：雌犬化、肉體調教、戰士的屈服。
*   **心理路徑**：正面挑戰主角 -> 體會到實力絕對差距 -> 在肉體快樂中感到罪惡 -> 承認自己渴望被強者支配的本能。

## 4. 帝國首席法師 - 薇薇安 (Vivian)
*   **稱號**：求知的魅魔（轉化前為人類）
*   **外貌**：紫髮波浪捲，眼鏡娘，黑絲與學士袍，巨乳。
*   **性格**：理智、冷靜、對未知知識有病態的渴求。
*   **關鍵詞**：知識換取身體、精神操控、催眠。
*   **心理路徑**：研究主角的力量 -> 交易身體換取深淵知識 -> 理智被快感覆寫 -> 成為主動獻策的軍師。

## 5. 異國舞姬/刺客 - 娜蒂亞 (Nadia)
*   **稱號**：致命的蜜糖
*   **外貌**：黑皮，異域風情舞孃裝扮，身上有金飾與紋身。
*   **性格**：狡猾、投機主義、善於偽裝。
*   **關鍵詞**：假戲真做、求饒、所有物化。
*   **心理路徑**：假意投誠刺殺 -> 陰謀被識破 -> 恐懼死亡 -> 發現作為寵物比作為殺手更安全 -> 徹底依賴。
