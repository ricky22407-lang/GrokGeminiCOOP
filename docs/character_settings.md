# 五位女主角詳細設定

所有角色：成年成熟女性，身材豐滿、氣場強勢，立繪強調曲線與威嚴。

1. 絕對忠誠型（王座近侍）
   - 名稱：薇爾緹（Velti）
   - 外貌：銀長髮、冷豔軍裝、高挑軍事體型。
   - 性格：沉默執行者，從一開始就渴望命令。
   - 心理路徑：奉獻 → 儀式化臣服 → 主動要求標記。
   - 關鍵詞：奉獻、儀式、宣誓。
   - H定位：主動跪下張開雙腿，語線「請用您的精液標記我的子宮，作为永遠的誓約」。

2. 理性交易型（策略顧問）
   - 名稱：塞蕾娜（Serena）
   - 外貌：黑長髮、知性眼鏡、緊身長袍凸顯胸臀。
   - 性格：冷靜計算，一開始視主角為合作對象。
   - 心理路徑：利益交換 → 發現掌控逆轉 → 自願獻上身體換取更高權力。
   - 關鍵詞：交換、利益、掌控感逆轉。
   - H定位：從後入式插入時低語「我的理性告訴我，這是最優解……請射進來，讓我徹底屬於您」。

3. 崇拜信仰型（宗教代表）
   - 名稱：艾莉婭（Elia）
   - 外貌：金髮、聖潔長袍、優雅豐滿身材。
   - 性格：精神潔癖，把主角視為唯一真理。
   - 心理路徑：神格化 → 自我消解 → 用身體證明信仰。
   - 關鍵詞：信仰、神格化、自我消解。
   - H定位：傳教士體位眼神直視，喘息間宣告「您的精液是聖水，請灌滿我的子宮，讓我成為您的聖殿」。

4. 抗拒轉化型（前敵對者）
   - 名稱：卡米拉（Camilla）
   - 外貌：紅短髮、女王裝甲、強勢驕傲體型。
   - 性格：高傲戰士，從否定主角到被迫承認。
   - 心理路徑：敗北 → 動搖 → 自尊重塑為歸屬。
   - 關鍵詞：敗北、承認、重塑自尊。
   - H定位：被壓制後強制高潮，語線「我……承認您是我的主人，請用您的肉棒懲罰我的傲慢，直到我乞求內射」。

5. 扭曲依附型（情報／暗殺系）
   - 名稱：莉絲（Lys）
   - 外貌：紫髮、暴露皮衣、危險妖豔身材。
   - 性格：情緒不穩，愛與恐懼交織。
   - 心理路徑：依存 → 控制需求 → 錯位救贖。
   - 關鍵詞：依存、控制、救贖錯位。
   - H定位：騎乘位主動扭腰，語線「只有您的精液能平息我的瘋狂……請射進最深處，讓我永遠無法離開」。
