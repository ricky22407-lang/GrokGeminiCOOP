# 世界觀設定：艾瑞西亞 (Erysia) 的黃昏

## 核心概念
「神權的終結，魔權的絕對支配。」

## 地理與勢力
1.  **聖光都城 (Lumina)**
    *   簡介：人類最後的堡壘，由教會高壓統治。表面聖潔，實則腐敗。
    *   現狀：即將成為主角的第一個征服目標。
2.  **永恆尖塔 (The Spire)**
    *   簡介：主角的大本營，一座反物理法則的黑色高塔，懸浮於虛空裂縫之上。
    *   功能：魔力源頭，囚禁與轉化俘虜的場所。
3.  **精靈之森 (Verdant)**
    *   簡介：排外的高等精靈領地，擁有古老結界。
    *   戰略價值：豐富的魔力資源與稀有的魔法素材。

## 力量體系
*   **支配值 (Dominance)**：主角的核心能量。透過征服土地、擊潰強敵意志、以及與關鍵角色的性互動獲得。
*   **意志力 (Willpower)**：女主角的防禦數值。隨著劇情推進與H事件會逐漸轉化為**順從度 (Submission)**。
*   **深淵魔力**：主角獨有的力量，無效化常規魔法，能直接侵蝕靈魂。

## 歷史背景
三百年前，諸神封印了深淵。如今封印破碎，主角作為「虛空之王」甦醒。他不需要毀滅世界，他需要世界臣服於他的秩序之下。原有的國王與英雄，不過是舊時代的殘黨。
