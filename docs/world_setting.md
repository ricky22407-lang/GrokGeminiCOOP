# 世界觀設定

大陸名：埃爾多蘭（Eldoran）

表面秩序：
- 多個種族城邦割據：人類王國、精靈森域、獸人部落、矮人堡壘、異形地下城。
- 存在虛偽的聯盟、貿易、信仰體系（多神教、王權宣稱正義）。
- 實則腐敗：領主掠奪資源、神殿販賣赦免、戰爭僅為私利。

王座的本質：
- 非國家、非軍隊，而是一種絕對存在。
- 主角（王座）擁有「裁定權」：能重寫規則、標記歸屬、剝奪反抗者的意志力。
- 歸屬機制：
  - 主動宣誓者：獲得保護、力量提升、慾望被滿足。
  - 反抗者：被逐步消耗——資源流失、盟友背叛、身體與心智逐漸渴求支配。
- 目標：讓「被支配」成為最合理的選擇。世界不會被摧毀，只會被重塑為跪姿。

種族與征服重點：
- 所有外族女代表均為成熟女性，無幼態。
- 征服非武力，而是心理與慾望：讓她們親自獻上族群主權，交換個人歸屬。
- H場景定位：儀式化交合，強調姿態服從（如跪舔、暴露宣誓）、體液標記、語言宣告歸屬。

主角視角：
- 世界是棋盤，所有人都是可被標記的棋子。
- 支配即效率：屈服者獲得永續，不屈者被用盡。
