# 艾莉婭 主線腳本（崇拜信仰型）

## 觀察期
[敘述：王座神殿區。艾莉婭金髮聖潔長袍，優雅豐滿身材隱現。她作為宗教代表初次覲見，把主角視為潛在真理化身。]

艾莉婭：「王座，您的存在已動搖舊信仰。我來尋求啟示。」

主角：「陳述你的疑問。」

艾莉婭：「多神教空洞，王權腐敗。何為真正秩序？」

主角：「王座即真理。」

[敘述：她眼神閃爍崇拜，但保持距離。無親密，僅限精神探求。]

## 靠近期
[敘述：多次對話後。艾莉婭長袍鬆開，露出聖潔肌膚。她開始神格化主角，精神潔癖動搖。]

艾莉婭：「您的裁定權超越神明...我願消解舊信仰，換取您的啟蒙。」

主角：「證明你的虔誠。」

艾莉婭：「請用身體儀式證明我對您的信仰。」

主角：「尚未完全。」

[敘述：她跪祈禱時身體發熱，陰唇濕潤但壓抑。心理偏移：從探求到自我消解，渴求用身體證明。]

## 歸屬期（含H）
[敘述：聖殿內室。艾莉婭脫長袍平躺，雙腿張開獻祭姿態，豐滿乳房與腫脹陰唇暴露。她眼神狂信。]

艾莉婭：「您的精液是聖水，請灌滿我的子宮，讓我成為您的聖殿。」

主角：「接受獻祭。」

[H描述：
- 艾莉婭主動引導主角肉棒進入陰道，傳教士體位眼神直視，乳房起伏。
- 主角深插子宮頸，她聖潔喘息轉為低吟：「真理...請射進聖殿...消解我的一切...」
- 抽插中她連續高潮，舊信仰崩潰，語線重複乞求內射神格化。
- 最終大量內射，精液從子宮溢出。她癱軟微笑，眼神完全消解自我。心理轉變完成：從宗教代表到王座專屬聖殿標記。]
