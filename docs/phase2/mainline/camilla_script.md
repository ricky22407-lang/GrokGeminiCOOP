# 卡米拉 主線腳本（抗拒轉化型）

## 觀察期
[敘述：王座對峙廳。卡米拉紅短髮女王裝甲，強勢驕傲體型。她作為前敵對者初次被召見，眼神充滿挑釁。]

卡米拉：「王座？你不過是個自稱支配者的幻影。我的王國不會屈服。」

主角：「觀察你的抵抗。」

卡米拉：「試試看吧，我會讓你後悔。」

主角：「繼續。」

[敘述：她高傲站立，無親密。主角冷靜觀察她的動搖跡象。]

## 靠近期
[敘述：多次衝突後。卡米拉裝甲破損，呼吸急促。她開始動搖，驕傲開始崩裂。]

卡米拉：「...你的力量超出預料。但我絕不承認。」

主角：「證明你的抵抗。」

卡米拉：「我...會讓你付出代價。」

主角：「敗北已近。」

[敘述：她身體顫抖，陰部隱隱濕潤。心理偏移：從否定到動搖，開始質疑自尊。]

## 歸屬期（含H）
[敘述：戰敗殿室。卡米拉裝甲全脫，跪地雙手反綁，強勢身軀暴露，陰唇腫脹。她眼神從憤怒轉為屈服。]

卡米拉：「我……承認您是我的主人，請用您的肉棒懲罰我的傲慢，直到我乞求內射。」

主角：「重塑自尊。」
