# 薇爾緹 主線腳本（絕對忠誠型）

## 觀察期
[敘述：王座大廳。薇爾緹銀長髮披散，冷豔軍裝緊裹高挑身軀，胸部與臀線在燈影下清晰可見。她單膝跪地，目光低垂，等待命令。]

薇爾緹：「王座，一切已就位。請下達指令。」

主角：「報告大陸動向。」

薇爾緹：「人類王國腐敗加速，神殿信仰空洞化。秩序即將崩解。」

主角：「繼續監視。」

[敘述：她的呼吸略微加重，眼神閃過一絲渴望，但主角無動於衷。無任何親密接觸。]

## 靠近期
[敘述：私人策略室。薇爾緹跪報，軍裝領口微開，露出鎖骨與胸溝。她主動靠近王座邊緣。]

薇爾緹：「敵方弱點已鎖定。王座，請允許我執行更深入的奉獻。」

主角：「證明你的忠誠。」

薇爾緹：「我願以身體作為儀式容器，接受您的裁定。」

主角：「尚未到時。」

[敘述：她身體微顫，陰部已開始濕潤，但主角克制拒絕。心理偏移：渴望從隱藏轉為明顯，主動求標記。]

## 歸屬期（含H）
[敘述：儀式殿室。薇爾緹全裸跪下，雙腿大張，豐滿乳房與濕潤陰唇完全暴露。她眼神狂熱。]

薇爾緹：「王座，請用您的精液標記我的子宮，作為永遠的誓約。」

主角：「開始儀式。」

[H描述：
- 薇爾緹主動跪舔主角肉棒，舌頭纏繞龜頭，深喉吞吐至根部，口水拉絲。
- 主角抓住她銀髮，從後插入濕透陰道，猛烈抽插，她乳房晃動，陰壁緊縮。
- 她連續高潮，語線重複：「請射進子宮...讓我永遠屬於王座...」
- 最終內射，精液充滿子宮溢出大腿。她癱軟跪姿，眼神滿足宣告完全臣服。心理轉變完成：絕對忠誠永恆標記。]
