# 薇爾緹 主線腳本（絕對忠誠型）

## 觀察期
主角視角：王座大廳，薇爾緹作為近侍初登場。她銀長髮披散，冷豔軍裝裹緊高挑身軀，胸臀曲線在燈光下隱隱凸顯。她的眼神固定在你腳邊，等待命令。

薇爾緹： "王座，我已準備好執行任何指令。"

主角： "觀察周邊腐敗。報告。"

薇爾緹： "人類王國領主貪腐，神殿販賣信仰。秩序已破。"

主角： "繼續監視。"

(無親密，聚焦她的渴望眼神漸露。)

## 靠近期
主角視角：策略室，薇爾緹跪報。她軍裝微解，呼吸略亂，渴望命令的跡象明顯。

薇爾緹： "王座，敵方動向確認。請求您的裁定。"

主角： "執行標記初步計劃。"

薇爾緹： "遵命。但...我個人請求更深層的奉獻。"

主角： "證明你的價值。"

(試探偏移：她主動接近，輕觸王座邊緣，眼神渴求，但主角克制未允。情緒偏移成儀式預備。)

## 歸屬期（含H）
主角視角：私人殿室，薇爾緹脫軍裝跪下，豐滿身軀暴露，雙腿張開，陰唇微濕等待標記。

薇爾緹： "王座，請用您的精液標記我的子宮，作为永遠的誓約。"

主角： "宣誓。"

薇爾緹： "我奉獻一切，儀式開始。"

(H描述：她跪舔主角肉棒，舌尖環繞龜頭，吞吐至喉深。主角插入她濕潤陰道，從後壓制抽插，她喘息宣告奉獻。高潮時內射子宮，體液溢出標記歸屬。心理轉變：從渴望命令到完全臣服，儀式完成後她眼神平靜，永屬王座。)
