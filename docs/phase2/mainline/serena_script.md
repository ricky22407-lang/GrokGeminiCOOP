# 塞蕾娜 主線腳本（理性交易型）

## 觀察期
[敘述：王座會客廳。塞蕾娜黑長髮盤起，知性眼鏡後冷靜眼神，緊身長袍包裹豐滿胸臀。她作為策略顧問初次覲見，視主角為潛在盟友。]

塞蕾娜：「王座，我帶來情報與提案。合作可帶來互利。」

主角：「陳述條件。」

塞蕾娜：「我提供大陸勢力分析，您給予保護與資源。」

主角：「接受初步交易。」

[敘述：她理性分析腐敗格局，眼神無波瀾。無親密，僅限利益交換。]

## 靠近期
[敘述：多次會談後。塞蕾娜長袍微亂，呼吸隱隱加速。她開始察覺主角裁定權的絕對性。]

塞蕾娜：「交易進展順利。但您的力量超出預測...我需要更多保障。」

主角：「提出新條件。」

塞蕾娜：「我願增加個人貢獻，換取更高層級的歸屬。」

主角：「證明你的理性選擇。」

[敘述：她主動靠近，輕撫王座扶手，陰部已濕但強自壓抑。心理偏移：從純利益到發現掌控逆轉，開始渴求身體交易。]

## 歸屬期（含H）
[敘述：私人交易室。塞蕾娜脫長袍跪坐，豐滿身軀暴露，眼鏡後眼神動搖。她雙腿張開，陰唇腫脹等待。]

塞蕾娜：「我的理性告訴我，這是最優解……請射進來，讓我徹底屬於您。」

主角：「確認交易完成。」
