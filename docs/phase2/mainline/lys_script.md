# 莉絲 主線腳本（扭曲依附型）

## 觀察期
[敘述：王座陰影區。莉絲紫髮暴露皮衣，危險妖豔身材。她作為情報暗殺者初次現身，眼神混雜愛與恐懼。]

莉絲：「王座，我帶來情報。但你...讓我不安。」

主角：「陳述你的目的。」

莉絲：「合作，或毀滅。我無法決定。」

主角：「觀察你的選擇。」

[敘述：她情緒不穩，無親密。主角冷靜觀察她的扭曲跡象。]

## 靠近期
[敘述：秘密會面後。莉絲皮衣半解，呼吸紊亂。她開始依附主角，恐懼轉為渴望。]

莉絲：「你的存在控制了我...我需要更多。」

主角：「證明你的依附。」

莉絲：「請...用方式救贖我的瘋狂。」

主角：「證明。」

[敘述：她主動觸碰自己，陰部濕潤。心理偏移：從不穩到錯位救贖，渴求控制。]

## 歸屬期（含H）
[敘述：扭曲殿室。莉絲全裸騎乘，雙腿夾緊主角腰，豐滿身軀扭動。她眼神瘋狂依戀。]

莉絲：「只有您的精液能平息我的瘋狂……請射進最深處，讓我永遠無法離開。」

主角：「接受依附。」
